  4. Modèle de détection d'objet - Projet Tuteuré

    2021-2022

    Création d’une IA Capable d’identifier des objets sur des images dans le cadre d’une caméra de surveillance. Language utilisé : Python avec diverses librairies (TensorFlow, Keras, Numpy...)

    Lors de la deuxième année d’IUT, j’ai travaillé sur un modèle d’IA capable d’identifier des objets sur des images dans le cadre d’une caméra de surveillance. Dans une équipe de cinq, nous avons séparé le projet en deux principales parties : deux personnes responsables de l’identification et 3 de la segmentation d’images dont moi. En tant que chef du projet, j’ai dû avoir une vue globale du projet ainsi que sur les jalons. Les réseaux de neurones précédemment créés auraient dû permettre de relever les points d’attention d’une image et de les identifier dans le but de connaître les événements mais malheureusement nous avons manqué de temps pour la réalisation de ce projet.
